# tracing around

minimalist web app for practicing arabic character strokes. because learning should be hands-on.

## what it does

- displays svg templates of arabic characters
- lets you trace with touch/stylus
- shuffles through 28 characters
- keeps it simple

## tech stack

- vanilla js
- html canvas
- touch events api
- svg templates

## quick start

1. clone it
2. open index.html
3. start tracing
